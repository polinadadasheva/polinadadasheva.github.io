# Как залить это на GitHub и получить рабочую ссылку на сайт

## 1. Загрузить файлы
1. Открой свой репозиторий на github.com
2. Нажми "Add file" → "Upload files" (кнопка обычно справа сверху над списком файлов)
3. Перетащи туда файлы: `index.html`, `style.css`, `script.js` и папку `images` с фотографией внутри
4. Внизу страницы нажми зелёную кнопку "Commit changes"

## 2. Включить показ сайта (GitHub Pages)
1. В репозитории зайди в "Settings" (вкладка сверху)
2. В левом меню найди "Pages"
3. В разделе "Build and deployment" → "Branch" выбери `main` (или `master`) и папку `/ (root)`
4. Нажми "Save"
5. Подожди 1-2 минуты, обнови страницу — сверху появится ссылка на твой сайт вида
   `https://твой-логин.github.io/название-репозитория/`

## Важно про фотографию
В файле `index.html` есть картинка `images/hero-photo.png` — это маленькое фото
рядом со словом "в" в заголовке. Файла пока нет, его нужно:
1. Открыть нужный фрейм в Figma
2. Кликнуть правой кнопкой на эту фотографию → Export → PNG → Export
3. Назвать скачанный файл `hero-photo.png` и положить в папку `images`

Если не положить файл — на странице вместо фото будет просто пустое место,
сайт от этого не "сломается".
